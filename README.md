# esp32.atomic14.com — corrections

Spotted a wrong or missing spec, pinout, or other detail on
**[esp32.atomic14.com](https://esp32.atomic14.com)**?

👉 **[Open a correction report](https://github.com/atomic14/esp32-website-issues/issues/new?template=correction.yml)**

Every page on the site has a "⚑ Report an error on this page" link that brings you
here with the page already filled in. The site's data is compiled from official
Espressif datasheets — please include the datasheet and page number where you can,
so corrections can be verified against the primary source.
